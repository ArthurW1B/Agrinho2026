# Agrinho2026
Site do Agrinho de 2026

Agroforte e futuro sustentável: a relação entre produção e meio ambiente

Neste site pretendo mostrar os benefícios da Agricultura de Baixo Carbono (ABC), esse tema é bem complicado pois com a agricultura de hoje em dia há uma grande emissão de carbono e com esse site pretendo demonstrar e concientizar, também colocando soluções para esses problemas.

Caso esteja perdido use os botões logo abaixo do título para poder se direcionar corretamente (caso houver dúvida). Neste site foi utilizado a ferramenta Canva para criar a logo.
